• O Bodhi Linux AppCenter é uma maneira simplificada de encontrar óptimas aplicações para a Bodhi.
  • O AppCenter é um serviço online que o Bodhi Linux fornece, onde poderá encontrar todo o tipo de software para incrementar o seu novo sistema.
  • Também fornecemos o Synaptic e o apt para instalar uma grande quantidade de software do Ubuntu e dos nossos próprios repositórios.
  • Ainda precisa daquele software específico que não encontra na nossa coleção? Pergunte no nosso fórum e nós adicioná-lo-emos para si!
