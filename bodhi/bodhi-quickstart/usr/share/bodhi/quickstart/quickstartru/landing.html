Добро пожаловать и спасибо за выбор Bodhi Linux — системы с минималистичным рабочим столом!



Bodhi — это лёгкий дистрибутив Linux с менеджером рабочих столов Moksha.

Наша философия состоит в предоставлении минимальной базовой системы. Дополнительные программы в неё можно устанавливать по мере необходимости.


Для получения дополнительных программ и обновления системы Bodhi вам потребуется подключение к сети Интернет.

Если Вам нужна помощь в этом вопросе, пожалуйста, перейдите по ссылке: Подключение к сети Интернет.


Поддержание операционной системы в актуальном состоянии важно с точки зрения информационной безопасности.

Рекомендуемый способ получения обновлений состоит в том, чтобы открыть терминал и выполнить команду:

sudo apt update && sudo apt dist-upgrade


Применение брандмауэра тоже повышает безопасность системы. Bodhi использует itables и содержит Упрощённый Брандмауэр (Uncomplicated Firewall) ufw. По умолчанию он выключен, потому что он требуется не всем.

Чтобы его включить, откройте терминал и выполните команду:

sudo ufw enable

Предустановленные настройки (профиль) брандмауэра подходят для большинства случаев бытового использования компьютера.
