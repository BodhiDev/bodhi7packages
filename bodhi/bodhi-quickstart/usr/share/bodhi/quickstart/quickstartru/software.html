Установка программ


В большинстве случаев для установки программ в Bodhi Linux необходимо работающее соединение с сетью Интернет.

Пожалуйста, перейдите по этой ссылке, если вам нужна помощь Подключение к сети Интернет.

Обновление системы с помощью Terminology

Первое, что надо сделать после установки интернет-подключения, это обновить вашу систему.

Для этого есть несколько способов, но проще всего воспользоваться терминалом. Выполните в Terminology следующую команду:

sudo apt update && sudo apt dist-upgrade

Если вы установили новый вариант Bodhi с расширенной поддержкой аппаратуры HWE, то обновления ядра будут устанавливаться по умолчанию.

Но традиционной позицией Bodhi является политика "не форсировать обновление ядра". Поэтому пользователи не-HWE вариантов Bodhi, желающие получить самое новое ядро, должны явным образом выполнить команду установки этого мета-пакета:

sudo apt install linux-generic


Центр приложений Bodhi Linux

Команда Bodhi Linux создала Центр приложений для быстрой и простой установки большинства наиболее востребованных программ. И хотя вы не найдёте в Центре приложений все программы, которые вы можете установить в Bodhi Linux, вы обнаружите там те, которые мы считаем лучшими в каждой конкретной категории.

Вот краткая инструкция по установке программ из Центра приложений:

  • Найдите программу.

    Если вы уже знаете название программы, которую хотите установить, просто наберите его в строке поиска.
    В противном случае, просто просмотрите категории, представленные на главной странице.

  • Перейдите на страницу.

    Вне зависимости от того, каким образом была найдена программа, вы получите ссылку на страницу с описанием этой программы.
    У каждой программы есть своя страничка с её описанием и снимком экрана.

  • Установите её!

    Просто нажмите кнопку "Install Now" и введите ваш пароль пользователя системы Bodhi, когда когда появится соответствующий запрос. На этом — всё! Ваша программа со всеми зависимостями будет установлена.



Менеджер пакетов Synaptic

Менеджер пакетов Synaptic, который знаком всем пользователям операционных систем, основанных на Debian, предустановлен в Bodhi 6.0. Если у вас старая версия Bodhi, установите его нажатием на представленную ниже кнопку или воспользуйтесь командой sudo apt install synaptic.

Менеджер пакетов Synaptic открывает вам доступ ко всем программам, которые представлены в репозиториях Bodhi и Ubuntu.

Обратитесь к нашей документации Wiki для получения подробных сведений об использовании менеджера пакетов Synaptic.


Snap, Flatpak и AppImage

В результате усилий, направленных на устранение зависимости программ от используемых компьютерных платформ, были разработаны новые инструменты упаковки, распространения и установки программ. Наибольшего внимания из них заслуживают Snap, Flatpak и AppImage. По умолчанию они не поддерживаются в Bodhi.

Обратитесь к нашей документации Wiki для получения подробных сведений об использовании этих инструментов.


gDebi

gdebi — это простой инструмент для установки deb-файлов. Он позволяет устанавливать локально сохранённые deb-пакеты, разрешая и устанавливая все их зависимости.

gDebi предустановлен в последних версиях Bodhi. Если вы хотите установить его, нажмите на представленную ниже кнопку или используйте команду sudo apt install gdebi.


apt-get & dpkg (для опытных пользователей)

Инструменты командной строки apt, apt-get и dpkg работают "за кулисами" описанных выше способов установки программ. Опытные пользователи могут использовать их напрямую из командной строки, но описание этого выходит за рамки данной инструкции.

Опытные пользователи могут также собирать и устанавливать программы из их исходных текстов.
