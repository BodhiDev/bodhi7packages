Bem-vindo(a) e obrigado por experimentar o Bodhi Linux: o ambiente gráfico minimalista e iluminado do Linux!



Bodhi é uma distribuição Linux leve que utiliza o gestor de janelas Moksha.

A nossa filosofia é disponibilizar um sistema básico minimalista. Os utilizadores podem instalar qualquer software adicional que desejem.


Precisará de uma ligação à Internet estabelecida para fazer atualizações do sistema ou instalar software no Bodhi.

Siga esta ligação se precisar de assistência: Ligação à Internet.


As atualizações do sistema são importantes de uma perspetiva de segurança.

A forma recomendada de efetuar atualizações do sistema é abrir um terminal e executar o comando:

sudo apt update && sudo apt dist-upgrade


Uma firewall é também importante de uma perspetiva de segurança. O Bodhi Linux usa o iptables e inclui o Uncomplicated Firewall (ufw). Não é ativada por padrão, pois nem todos os utilizadores podem precisar.

Para a ativar, abra um terminal e execute o comando:

sudo ufw enable

A ufw tem um conjunto criterioso de definições padrão (perfil), que são boas para a grande maioria dos utilizadores domésticos.
