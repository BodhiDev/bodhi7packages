Instalar Software


Na maioria dos casos, desejará ter uma ligação à Internet estabelecida para instalar software no Bodhi Linux.

Siga esta ligação se necessitar de assistência Ligação à Internet.

Atualizar o sistema via Terminology

A primeira coisa a fazer após estabelecer uma ligação à Internet é atualizar o sistema.

Existem vários métodos para o fazer, mas o método mais simples é a utilização do terminal. Execute o seguinte comando no Terminology:

sudo apt update && sudo apt full-upgrade

Tradicionalmente, Bodhi tem a visão de "não forçar atualizações do kernel". Vamos mudar isso no Bodhi Linux 7.0 porque notamos que muitos usuários não estão atualizando kernels e as atualizações do kernel são importantes do ponto de vista da segurança.



Bodhi Linux AppCenter

A equipa do Bodhi Linux criou a AppCenter como uma forma muito rápida e simples de instalar muitas aplicações comuns. Embora não encontre todas as aplicações que pode instalar no Bodhi Linux na AppCenter você irá encontrar o que consideramos ser as "melhores" aplicações para cada categoria em particular.

Eis um tutorial rápido sobre como instalar aplicações na AppCenter:

  • Encontre a sua aplicação

    Se já souber o nome da aplicação que deseja instalar, basta escrevê-la na barra de pesquisa.
    Caso contrário, basta navegar através das categorias listadas na página principal.

  • Navegue para a página

    Qualquer que seja o método escolhido para encontrar a sua aplicação, ser-lhe-á apresentado uma ligação que leva a uma página com uma descrição da aplicação.
    Cada aplicação tem a sua própria página com uma descrição da aplicação e uma imagem da mesma.

  • Instale!

    Basta clicar no botão "Instalar agora!" e introduzir a sua palavra-passe quando lhe for pedida. É isso mesmo! A sua aplicação e todas as suas dependências serão automaticamente instaladas.



Gestor de Pacotes Synaptic

O Gestor de Pacotes Synaptic, comum a todas as distribuições Debian, está instalado no Bodhi 7.0 por padrão. Se instalou uma versão mais antiga do Bodhi, instale-o clicando no ícone abaixo ou use sudo apt install synaptic a partir da linha de comandos.

Com o Gestor de Pacotes Synaptic, tem um acesso a todas as aplicações nos repositórios Bodhi e Ubuntu

Veja a nossa Wiki para instruções detalhadas sobre a utilização do Gestor de Pacotes Synaptic.


Snap, Flatpak e AppImage

Num esforço para passar de métodos de empacotamento específicos da plataforma para métodos agnósticos de plataforma, foram desenvolvidas várias novas ferramentas de empacotamento, distribuição e instalação de software. Os mais notáveis são o Snap, Flatpak e AppImage. Estes não são suportados por padrão no Bodhi.

Veja a nossa Wiki para instruções detalhadas sobre a utilização destas ferramentas.


gDebi

O gdebi é uma ferramenta simples para instalar ficheiros ".deb". Permite-lhe instalar pacotes ".deb" locais, resolvendo e instalando todas as dependências.

O gDebi foi instalado por padrão em versões anteriores recentes do Bodhi. Se desejar instalá-lo, clique no ícone abaixo ou use o sudo apt install gdebi a partir da linha de comandos.


apt & dpkg (advanced)

As ferramentas de linha de comandos apt e dpkg são o "backend" para os métodos de instalação de aplicações acima referidos. Os utilizadores avançados são livres de utilizar estas ferramentas a partir da linha de comandos, mas isso está fora do âmbito deste documento.

Os utilizadores avançados podem também instalar a partir do código fonte.
