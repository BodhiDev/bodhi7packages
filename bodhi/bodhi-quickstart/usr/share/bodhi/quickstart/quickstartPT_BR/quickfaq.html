Perguntas mais frequentes


Muitos dos novos utilizadores do Bodhi Linux parecem ter as mesmas perguntas, por isso aqui estão as respostas a algumas delas:

O que é o Bodhi Linux?

O Bodhi Linux é um sistema operacional baseado em Linux construído sobre as versões de suporte a longo prazo do Ubuntu. O nosso objetivo é fornecer um sistema operacional estável, minimalista e fácil de usar. Temos uma nova versão major a cada 2 anos quando um novo Ubuntu LTS é lançado. Cada lançamento é suportado durante 5 anos sem que a base do Ubuntu LTS fique sem suporte.

Porque é que existem tão poucas aplicações instaladas?

Parte da filosofia do Bodhi Linux é sobre minimalismo e escolha do usuário. Sentimos que os nossos usuários são suficientemente inteligentes para escolher as aplicações que desejam, e fornecemos uma forma fácil de fazê-lo no AppCenter. Também sentimos que é mais fácil e melhor para si construir o sistema que quer desde o início, em vez de ter de remover muitas aplicações que não quer e/ou precisa e depois instalar as que quer/precisa.

Você pode aprender mais sobre instalação de software no Bodhi Linux a partir daqui.

O que é o Moksha?

O Moksha é uma derivação do ambiente gráfico do Enlightenment DR17 mantido pela equipe do Bodhi Linux. O Moksha concentra-se em permanecer leve, oferecendo ao mesmo tempo um elevado nível de opções de personalização.
