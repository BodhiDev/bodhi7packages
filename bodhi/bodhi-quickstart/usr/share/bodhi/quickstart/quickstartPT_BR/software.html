Instalar Software


Na maioria dos casos, você desejará ter uma conexão à Internet estabelecida para instalar software no Bodhi Linux.

Veja esta página se necessitar de assistência: Conectar à Internet.

Atualizar o sistema via Terminology

A primeira coisa a fazer após estabelecer uma conexão à Internet é atualizar o sistema.

Existem vários métodos para fazê-lo, mas o método mais simples é a utilização do terminal. Execute o seguinte comando no Terminology :

sudo apt update && sudo apt full-upgrade

Tradicionalmente, Bodhi tem a visão de "não forçar atualizações do kernel". Vamos mudar isso no Bodhi Linux 7.0 porque notamos que muitos usuários não estão atualizando kernels e as atualizações do kernel são importantes do ponto de vista da segurança.



Bodhi Linux AppCenter

A equipe do Bodhi Linux criou o AppCenter como uma forma muito rápida e simples de instalar muitas aplicações comuns. Embora não encontre todas as aplicações que se pode instalar no Bodhi Linux, na AppCenter, você irá encontrar o que consideramos ser as "melhores" aplicações para cada categoria em particular.

Eis um tutorial rápido sobre como instalar aplicações na AppCenter:

  • Encontre a sua aplicação.

    Se já souber o nome da aplicação que deseja instalar, basta escrevê-la na barra de pesquisa.
    Caso contrário, basta navegar através das categorias listadas na página principal.

  • Navegue para a página.

    Qualquer que seja o método escolhido para encontrar a sua aplicação, você chegará a um link que leva a uma página com uma descrição da aplicação.
    Cada aplicação tem a sua própria página com uma descrição e uma imagem da mesma.

  • Instale!

    Basta clicar no botão "Instalar agora" e introduzir a sua senha quando lhe for pedida. É isso mesmo! A sua aplicação e todas as suas dependências serão automaticamente instaladas.



Gerenciador de Pacotes Synaptic

O Gerenciador de Pacotes Synaptic, comum a todas as distribuições Debian, está instalado desde o Bodhi 6.0 por padrão. Se instalou uma versão mais antiga do Bodhi, instale-o clicando no ícone abaixo ou use sudo apt install synaptic a partir da linha de comando.

Com o Gerenciador de Pacotes Synaptic, você tem acesso a todas as aplicações nos repositórios Bodhi e Ubuntu.

Veja a nossa Wiki para instruções detalhadas sobre a utilização do Gerenciador de Pacotes Synaptic.


Snap, Flatpak e AppImage

Num esforço para passar de métodos de empacotamento específicos da plataforma para métodos agnósticos de plataforma, foram desenvolvidas várias novas ferramentas de empacotamento, distribuição e instalação de software. Os mais notáveis são o Snap, Flatpak e AppImage. Estes não são suportados por padrão no Bodhi.

Veja a nossa Wiki para instruções detalhadas sobre a utilização destas ferramentas.


gDebi

O gDebi é uma ferramenta simples para instalar arquivos ".deb". Permite-lhe instalar pacotes ".deb" locais, resolvendo e instalando todas as dependências.

gDebi foi instalado como padrão em todas as versões passadas recentes do Bodhi. Se desejar instalá-lo clique no ícone abaixo ou use sudo apt install gdebi a partir da linha de comando.


apt & dpkg (avançado)

As ferramentas de linha de comando apt, apt-get e dpkg são o "backend" para os métodos de instalação de software acima referidos. Os usuários avançados são livres para utilizar estas ferramentas a partir da linha de comando, mas isto está além do âmbito deste documento.

Os usuários (ainda mais) avançados podem também instalar a partir do código fonte.
