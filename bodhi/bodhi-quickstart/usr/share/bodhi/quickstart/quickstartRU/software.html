Установка программ


В большинстве случаев для установки программ в Bodhi Linux необходимо работающее соединение с сетью Интернет.

Перейдите по этой ссылке, чтобы узнать, как настроить Подключение к сети Интернет.

Обновление системы с помощью Terminology

Первое, что надо сделать после подключения к сети Интернет, это обновить вашу систему.

Для этого есть несколько способов, но проще всего воспользоваться терминалом. Выполните в Terminology следующую команду:

sudo apt update && sudo apt full-upgrade

Традиционная позиция Bodhi, — "не устанавливать обновления ядра" по умолчанию. Это изменится в Bodhi Linux 7.0, потому что многие пользователи не обновляют ядро самостоятельно, а обновления ядра важны с точки зрения информационной безопасности.



Центр приложений Bodhi Linux

Команда Bodhi Linux создала Центр приложений для быстрой и простой установки наиболее востребованных программ. И хотя вы не найдёте в Центре приложений все программы, которые можно установить в Bodhi Linux, вы найдёте там те, которые мы считаем «лучшим выбором» в каждой конкретной категории.

Вот краткая инструкция по установке программ из Центра приложений:

  • Найдите программу

    Если вы уже знаете название нужной программы, просто наберите его в строке поиска.
    В противном случае просмотрите категории, представленные на главной странице Центра приложений.

  • Перейдите на её страницу

    Вне зависимости от того, как была найдена программа, вы получите ссылку на посвящённую ей страницу.
    У каждой программы есть собственная страница с её описанием и снимками экрана.

  • Установите программу!

    Просто нажмите кнопку "Install Now" и введите ваш пароль пользователя Bodhi Linux, когда появится соответствующий запрос. На этом — всё! Ваша программа и все её зависимости будут установлены автоматически.



Менеджер пакетов Synaptic

Менеджер пакетов Synaptic, известный пользователям всех дистрибутивов, основанных на Debian, предустановлен в Bodhi 7.0. Если вы пользуетесь старой версией Bodhi, установите его, нажав на размещённую ниже кнопку, или воспользуйтесь командой sudo apt install synaptic.

Менеджер пакетов Synaptic открывает доступ ко всем программам, имеющимся в репозиториях Bodhi и Ubuntu.

Обратитесь к нашей документации Wiki для получения подробных сведений об использовании менеджера пакетов Synaptic.


Snap, Flatpak и AppImage

В результате усилий, направленных на устранение зависимости программ от компьютерных платформ, были разработаны новые инструменты упаковки, распространения и установки программ. Из них наибольшего внимания заслуживают Snap, Flatpak и AppImage. По умолчанию они не поддерживаются в Bodhi.

Обратитесь к нашей документации Wiki для получения подробных сведений об использовании этих инструментов.


gDebi

Для установки deb-файлов есть простой инструмент gdebi. Он позволяет устанавливать сохранённые локально deb-пакеты, разрешая и устанавливая все зависимости.

Инструмент gDebi предустановлен в последних версиях Bodhi. Если в вашей системе его нет, то установить его можно нажатием на расположенную ниже кнопку или с помощью команды sudo apt install gdebi.


apt & dpkg (для опытных пользователей)

Инструменты командной строки apt и dpkg работают «за кулисами» описанных выше способов установки программ. Опытные пользователи могут работать с ними непосредственно, но описание этого выходит за рамки данного руководства.

Опытные пользователи могут также собирать и устанавливать программы из исходных текстов.
