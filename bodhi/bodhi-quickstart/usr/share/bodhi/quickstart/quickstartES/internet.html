Conectándose a Internet




Bodhi facilita la conexión a Internet oa su red local con el uso de Network Manager (nm-applet).


Verá un ícono similar a este en su panel.


Todo lo que tienes que hacer es hacer clic izquierdo sobre él. Luego elija el tipo de conexión de la lista disponible. Network Manager es capaz de manejar muchos tipos de conexiones de red, incluidas ethernet, banda ancha móvil, inalámbrica, VPN y DSL. Muchas conexiones se configurarán automáticamente; algunos necesitarán datos del usuario, como una contraseña.

Después de unos momentos, una conexión debería establecerse.

las conexiones se pueden editar haciendo clic derecho en el ícono, enseguida Editar las conexiones para abrir el administrador de configuración. Además, observe que el ícono cambia su apariencia una vez que se establece una conexión. Una característica común que se habilita automáticamente es que la conexión en cuestión se conecte automáticamente tan pronto como se detecte.

Los problemas de conexión pueden ser notoriamente difíciles de diagnosticar, especialmente en un sistema que no puede establecer ninguna conexión. Si estos pasos fallan, el mejor consejo que podemos dar es utilizar un sistema que establezca una conexión (o usar una conexión por cable si falla la conexión inalámbrica) y unirse a nosotros en Discord o en los foros de Bodhi. Consulte la página de Recursos.
