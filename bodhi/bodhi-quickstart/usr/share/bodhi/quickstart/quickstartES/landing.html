Bienvenido y gracias por probar Bodhi Linux: ¡el escritorio Linux iluminado y minimalista!



Bodhi es una distribución ligera de Linux que utiliza el gestor de ventanas Moksha.

Nuestra filosofía es proporcionar un sistema base mínimo. Los usuarios pueden instalar cualquier software adicional que deseen.


Necesitará una conexión a Internet para realizar actualizaciones del sistema o instalar software en Bodhi.

Por favor siga este enlace si necesita ayuda: Connectándose a Internet.


Las actualizaciones del sistema son importantes desde una perspectiva de seguridad.

La forma preferida de realizar actualizaciones del sistema es abrir una terminal y ejecutar el comando:

sudo apt update && sudo apt full-upgrade


Un firewall también es importante desde una perspectiva de seguridad. Bodhi usa iptables e incluye un cortafuegos sin complicaciones (ufw). No está habilitado de forma predeterminada, ya que es posible que no todos los usuarios lo necesiten.

Para habilitarlo, abra una terminal y ejecute el comando:

sudo ufw enable

Ufw tiene un conjunto razonable de configuraciones predeterminadas (perfiles), que están bien para la gran mayoría de los usuarios domésticos.
