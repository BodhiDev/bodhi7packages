Instalando Software

En la mayoría de los casos, querrá tener una conexión a Internet para instalar el software en Bodhi Linux.

Siga este enlace si necesita ayuda para conectarse a Internet: Conectando a Internet.

Actualización del sistema a través de Terminology

Lo primero que debe hacer después de establecer una conexión a Internet es actualizar su sistema.

Hay varios métodos para lograr esto, pero el método más simple es usar la terminal. Ejecute el siguiente comando en Terminology:

sudo apt update && sudo apt full-upgrade

Tradicionalmente, Bodhi ha adoptado la postura de no forzar las actualizaciones del kernel. Cambiaremos esto en Bodhi Linux 7.0 porque hemos notado que muchos usuarios no están actualizando los kernels y las actualizaciones del kernel son importantes desde el punto de vista de la seguridad.



Bodhi Linux AppCenter

El equipo de Bodhi Linux ha creado el AppCenter como una forma muy rápida y sencilla de instalar muchas aplicaciones comunes. Si bien no encontrará en el AppCenter todas las aplicaciones que se pueden instalar en Bodhi Linux, encontrará lo que consideramos las aplicaciones "mejores de su clase" para cada categoría en particular.

Aquí hay un tutorial rápido sobre cómo instalar aplicaciones en el AppCenter:

  • Encuentra tu aplicación

    Si ya conoce el nombre de la aplicación que desea instalar, simplemente escríbalo en la barra de búsqueda.
    De no ser así, simplemente navegue por las categorías enumeradas en la página principal.

  • Navegar a la página

    Cualquiera que sea el método que elija para encontrar su aplicación, se le presentará un enlace que lleva a una página con una descripción de la aplicación.
    Cada aplicación tiene su propia página con una descripción y una captura de pantalla.

  • ¡A instalar!

    Simplemente haga clic en el botón "Instalar ahora" e ingrese su contraseña cuando se le solicite. ¡Eso es todo! Su aplicación y todas sus dependencias se instalarán automáticamente.



Synaptic Package Manager

El administrador de paquetes Synaptic, común a todas las distribuciones basadas en Debian, está instalado en Bodhi 7.0 de forma predeterminada. Si tiene una versión anterior de Bodhi, instálelo haciendo clic en el ícono mostrado a continuación, o use sudo apt install synaptic desde la línea de comandos.

Con el Administrador de paquetes Synaptic, tiene acceso a todas las aplicaciones en los repositorios de Bodhi y Ubuntu.

Por favor lea nuestro Wiki para obtener instrucciones completas acerca del uso del Administrador de paquetes Synaptic.


Snap, Flatpak y AppImage

En un esfuerzo por alejarse de los métodos de empaquetamiento específicos de plataforma a métodos agnósticos, se han desarrollado varias herramientas nuevas para empaquetar, distribuir e instalar software. Las más destacadas son Snap, Flatpak y AppImage. Estas no son compatibles de forma predeterminada con Bodhi.

Por favor lea nuestro Wiki para obtener instrucciones completas sobre el uso de estas herramientas.


gDebi

gDebi es una herramienta sencilla para instalar archivos deb. Le permite instalar paquetes deb locales, resolviendo e instalando todas las dependencias.

gDebi se instalaba de forma predeterminada en versiones anteriores recientes de Bodhi. Si desea instalarlo, haga clic en el ícono a continuación o use sudo apt install gdebi desde la línea de comandos.


apt y dpkg (avanzado)

Las herramientas de línea de comandos apt and dpkg son el "backend" de los métodos anteriores de instalación de aplicaciones. Los usuarios avanzados pueden usar estas herramientas desde la línea de comandos, pero esto va más allá del alcance de este documento.

Los usuarios avanzados también pueden instalar desde código fuente.
