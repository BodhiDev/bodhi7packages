Mit dem Internet verbinden




Bodhi macht es dem Nutzer durch die Verwendung von Network Manager (nm-applet) leicht, eine Verbindung mit dem Internet oder einem lokalen Netzwerk herzustellen.


Ein Symbol, das so oder ähnlich aussieht, sollte in der Modulablage zu sehen sein.


Man muss nur einen Linksklick darauf ausführen und dann die gewünschte
Verbindungsart aus der Liste der verfügbaren Verbindugen aus wählen.
Network Manager kann viele Arten von Netzwerkverbindungen verwalten,
inklusive Ethernet, mobiles Breitband, WLAN, VPN und DSL.
Einige davon werden automatisch konfiguriert, anderen benötigen
Eingaben des Nutzers, z.B. ein Passwort.

Nach einigen Momenten sollte die Verbindung hergestellt sein.

Verbindungen können danach bearbeitet werden, indem man auf das Symbol klickt und so Network Manager auswählt und dann Bearbeiten anklickt um die Einstellungsverwaltung zu öffnen. Sobald eine Verbindung hergestellt ist, ändert das Symbol sein Aussehen. Eine häfige Einstellung, die automatisch aktiviert ist, ist das automatische Herstellen der Verbindung, sobald das entsprechende Netzwerk erkannt wurde.

Verbindungsprobleme sind bekannt dafür, schwierig zu diagnostizieren zu sein, besonders auf einem System, das gar keine Verbindungen herstellen kann. Wenn die obigen Schritte fehlschlagen, ist der beste Ratschlag, den wir geben können, ein System zu nehmen, das eine Verbindung herstellen kann (oder z.B. eine kabelgebundene zu nutzen, wenn das WLAN nicht funktioniert) und uns auf Discord oder dem Bodhi Forum zu besuchen. Siehe die Seite Weiterführende Informationen.
