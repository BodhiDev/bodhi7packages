Willkommen und danke für das Ausprobieren von Bodhi Linux: die minimalistische Erleuchtete Linux-Distribution!



Bodhi ist eine leichtgewichtige Linux Distribution, die den Moksha Fenstermanager nutzt.

Unsere Philosophie ist es, ein minimales Grundsystem bereitzustellen. Nutzer können zusätzliche Programme nach Belieben installieren.


Eine Internetverbindung wird benötigt, um das System zu aktualisieren oder Programme unter Bodhi zu installieren

Für Hilfe zum Herstellen einer Internetverbindung bitte diesem Link folgen: Mit dem Internet verbinden..


Systemaktualisierung sind vom Sicherheitsaspekt her wichtig.

Vorzugsweise werden Systemaktualisierung in der Kommandozeile vorgenommen, indem folgenden Befehl ausführt:

sudo apt update && sudo apt dist-upgrade


Eine Firewall ist ebenfalls wichtig vom Sicherheitsaspekt her. Bodhi nutzt iptables und enthält die Unkomplizierte Firewall (ufw). Sie ist nicht standardmäßig aktiviert, weil sie eventuell nicht von allen Nutzern benötigt wird.

Zur Aktivierung bitten den folgenden Befehl in der Kommandozeile ausführen:

sudo ufw enable

UFW hat ein vernüftiges Set von Standardeinstellungen (Profil), die für die allermeisten Privatnutzer in Ordnung sein dürften.
